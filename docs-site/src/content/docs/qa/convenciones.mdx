## Requisitos
- Claros, verificables y medibles.
- Enfocados al comportamiento observable.

## Escenarios de prueba
- Precondiciones
- Pasos
- Resultado esperado

## Ejemplo
- Requisito: El usuario puede crear una tarjeta con titulo obligatorio.
- Escenario:
  1. Abrir tablero con permisos.
  2. Pulsar "Nueva tarjeta".
  3. Introducir titulo y guardar.
  4. La tarjeta aparece en la columna inicial.
