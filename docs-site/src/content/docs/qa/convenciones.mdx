---

## Requisitos
- Claros, verificables y medibles.
- Enfocados al comportamiento observable.

## Escenarios de prueba
- Precondiciones
- Pasos
- Resultado esperado

## Regresion
- Marca los tests de regresion con `@regression` en el titulo.
- Estos tests se ejecutan siempre en `master` con Playwright.

## Specs y componentes
- Cada spec debe completar **Componente principal** en la plantilla.
- Para listar specs por componente: `node scripts/specs-by-component.mjs`

## Ejemplo
- Requisito: El usuario puede crear una tarjeta con titulo obligatorio.
- Escenario:
  1. Abrir tablero con permisos.
  2. Pulsar "Nueva tarjeta".
  3. Introducir titulo y guardar.
  4. La tarjeta aparece en la columna inicial.
