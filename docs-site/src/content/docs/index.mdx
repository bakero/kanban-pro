Bienvenido a la documentacion interactiva de Kanban Pro.

Aqui encontraras:
- Un arbol de funcionalidades de la app y la consola de administracion.
- Requisitos por funcionalidad.
- Escenarios de prueba para preview y produccion.
- El enlace a la ultima ejecucion registrada de cada prueba.

## Accesos rapidos
- [App](./app)
- [Consola de administracion](./admin)
- [QA](./qa/ejecuciones)

## Como mantener el contenido
Cada pagina de funcionalidad sigue el mismo esquema:
- Objetivo
- Funcionamiento
- Reglas de negocio
- Requisitos
- Escenarios de prueba
- Ultima ejecucion

Actualiza el bloque de "Ultima ejecucion" con el link al ultimo run de CI o al reporte Playwright.
