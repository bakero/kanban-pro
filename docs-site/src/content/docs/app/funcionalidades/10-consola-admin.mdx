## Objetivo
Ofrecer una consola global para el super admin.

## Funcionamiento
- Endpoint separado /admin.html.
- Gestion de empresas, usuarios, proyectos y tableros.
- Aprobacion centralizada de mejoras.

## Reglas de negocio
- Acceso exclusivo para el email super admin.

## Requisitos
- Debe cargar datos filtrados por empresa.

## Escenarios de prueba
1. Acceso con super admin y carga de empresas.
2. Aprobar mejora y validar estado ai_pending.
