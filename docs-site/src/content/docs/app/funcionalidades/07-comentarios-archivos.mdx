## Objetivo
Registrar comentarios y archivos asociados a una tarjeta.

## Funcionamiento
- Anadir comentarios con historial.
- Registrar adjuntos por nombre de fichero.

## Reglas de negocio
- Los comentarios quedan asociados a la tarjeta.

## Requisitos
- El historial debe mostrar comentarios en orden temporal.

## Escenarios de prueba
1. Anadir comentario y validar persistencia.
2. Adjuntar archivo y verificar listado.
