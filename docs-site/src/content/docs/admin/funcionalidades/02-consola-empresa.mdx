## Objetivo
Permitir a admins de empresa gestionar recursos propios.

## Funcionamiento
- Acceso restringido a company_admin.
- Secciones limitadas a datos de la empresa.

## Reglas de negocio
- Las acciones quedan registradas en logs.

## Requisitos
- No puede acceder a catalogo global.

## Escenarios de prueba
1. Acceso con rol company_admin y validacion de datos.
