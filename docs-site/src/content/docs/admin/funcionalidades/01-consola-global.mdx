## Objetivo
Gestionar empresas, usuarios, proyectos, tableros y trabajos.

## Funcionamiento
- Acceso exclusivo por email super admin.
- Paneles por seccion con datos por empresa.

## Reglas de negocio
- Solo super admin puede aprobar mejoras y editar catalogo global.

## Requisitos
- Debe cargar datos de todas las empresas de forma segura.

## Escenarios de prueba
1. Acceso con super admin y navegacion entre secciones.
