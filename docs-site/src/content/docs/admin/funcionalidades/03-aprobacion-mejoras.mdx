description: Flujo de aprobacion hacia IA.
---

## Objetivo
Controlar que mejoras pasan a estado ai_pending.

## Funcionamiento
- El super admin aprueba mejoras desde la consola.
- La mejora cambia a estado ai_pending.

## Reglas de negocio
- Usuarios normales no pueden aprobar.

## Requisitos
- La lista debe reflejar el estado actualizado.

## Escenarios de prueba
1. Aprobar mejora y verificar estado ai_pending.
