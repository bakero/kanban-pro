# Kanban Pro Docs

Sitio de documentacion interactiva basado en Astro Starlight.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Despliegue en Vercel
- Crear un proyecto en Vercel apuntando a `docs-site` como Root Directory.
- Comando de build: `npm run build`
- Output: `dist`

El enlace de produccion se puede copiar en `VITE_DOCS_URL` y `VITE_ADMIN_DOCS_URL`.
